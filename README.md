# Dumsor Agent — GitLab Transcend Hackathon 2026

An AI agent that predicts load-shedding schedules for Ghanaian communities.

## Problem
Millions of Ghanaians face unpredictable power outages daily. Businesses lose revenue. Students can't study. Hospitals rely on generators.

## Solution
A CLI tool that takes a neighborhood name and returns predicted load-shedding times based on local utility patterns.

## Features
- Schedule lookup for major Ghanaian cities and neighborhoods
- Report saving with timestamps
- Interactive menu interface
- Covers Accra, Kumasi, Tamale, Cape Coast, Takoradi and metro areas

## Run
python dumsor_agent.py
